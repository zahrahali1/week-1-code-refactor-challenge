# <Code Refactor Challenge 1>

## Description

In this project we as a marketing agency want to follow accessibility standards by improving the code without changing the output of the page to ensure that people with disabilities can access the website.

I've added some semantic html elements such as header and footer to provide clear structure so I changed "div class" which is a more older style html so I chnaged a class selector to an element selector.
I've also added alt attributes for image elements with a descriptive text next to the images explaining what it is.
I've made the title more descriptive in the HTML file to make the the users more understanding.
The HTML elements follow a logical strucure in styling and positioning.
